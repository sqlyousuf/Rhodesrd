# Rhodes Rd Expenses

A simple expense tracking app for iPhone.

## Features

- Add expenses with work done and amount
- View total expenses
- Edit and delete entries
- Data persists on device

## How to Run

1. Install Expo Go on your iPhone from the App Store.
2. Scan the QR code displayed when running the app.
3. Or run on web: `npm run web` and open http://localhost:8081

## Development

- `npm start` to start Expo dev server
- `npm run ios` for iOS simulator (requires macOS)
- `npm run android` for Android emulator
- `npm run web` for web version
